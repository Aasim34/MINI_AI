# Customer Churn Prediction

A full-stack application that predicts customer churn using machine learning algorithms, helping businesses identify at-risk customers and take proactive measures.

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Setup Environment
```bash
cp .env.local.example .env.local
```
Edit `.env.local` with your database URL and other settings.

### 3. Setup Database
```bash
npx prisma db push
npx prisma generate
```

### 4. Run Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/pages/` - Next.js pages and API routes
- `src/components/` - React components
- `src/styles/` - CSS and styling
- `src/lib/` - Utility functions
- `prisma/` - Database schema

## Features

- Data Analysis Dashboard
- Churn Prediction Model
- Customer Segmentation
- Real-time Alerts for High-Risk Customers
- Reporting and Analytics

## API Endpoints

- `GET /api/customers` - Retrieve the list of customers
- `GET /api/customers/:id` - Get customer details
- `POST /api/predictions` - Submit churn prediction for a customer
- `GET /api/predictions/:id` - Get prediction results for a specific customer

## Tech Stack

- **Frontend**: Next.js 14, React 18, Tailwind CSS 3
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM
